Installiere die CLI

Vorraussetzungen

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install SkyCompute-cli

NPM

$ npm install SkyCompute-cli

Yarn

$ yarn install SkyCompute-cli
